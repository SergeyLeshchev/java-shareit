ShareIt — бэкенд-сервис для обмена вещами. Пользователи могут брать в аренду вещи, управлять бронированиями и оставлять отзывы.


Технологический стек

Backend: Java 21, Spring Boot, Spring Data JPA (Hibernate)

База данных: PostgreSQL

Архитектура: Микросервисная (Gateway + Main Service)

Контейнеризация: Docker, Docker Compose

Клиент-сервер: RestTemplate, HTTP-заголовки (X-Sharer-User-Id)

-------------------

Ключевые функции

Управление вещами: CRUD, поиск по описанию

Бронирование: Создание с проверкой доступности, подтверждение владельцем, фильтрация по статусам

Запросы вещей: Создание и просмотр запросов

Комментарии: Комментарии от завершивших аренду пользователей

-------------------

Микросервисная архитектура (2 модуля):

Gateway:

Единая точка входа

Валидация входных данных

Маршрутизация запросов на основной сервис через RestTemplate

Добавление заголовка X-Sharer-User-Id для идентификации

Server:

Вся бизнес-логика

Работа с БД через Spring Data JPA

Чистая пакетная структура по доменам
